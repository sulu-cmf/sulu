# Security Policy

## Supported Versions

| Version   | Supported          |
| --------- | ------------------ |
| >= 2.4.0  | :white_check_mark: |
| 2.0 - 2.3 | :x:                |
| 1.6.x     | :white_check_mark: |
| < 1.6     | :x:                |

## Reporting a Vulnerability

You can contact us for security related issues by using [security@sulu.io](mailto:security@sulu.io).
